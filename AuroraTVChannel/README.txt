Aurora TV Channel - Expo project
Primary color: #2196F3

Instructions:
1. Extract this folder.
2. Install dependencies:
   npm install
3. Login to Expo:
   npx expo login
4. Build APK (using EAS):
   npx eas build -p android --profile preview
5. Download the generated .apk from the Expo build page and install on your Android device.

Note: Replace placeholder assets in /assets with real images for icon/splash/adaptive-icon/favicon.
